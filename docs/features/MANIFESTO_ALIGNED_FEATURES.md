# 20 Manifesto-Aligned Features

## Your Core Principles
- Documents, not apps
- Static-first, zero JS on content
- AI-assisted, human-controlled
- Platform-agnostic
- Performance budgets enforced
- Accessibility by default
- Machine-readable (AI agents + SEO)

---

## 20 Features That Align

### Tier 1: AI-Agent Optimization (The Future Web)

1. **RSS-to-Everything Syndicator**
   - Auto-post to Dev.to, Medium, Hashnode, LinkedIn
   - Canonical links back to your site
   - Platform-agnostic content distribution
   
2. **Vector Embeddings Generator**
   - Generate embeddings for all content
   - Semantic search without JS
   - AI agents can find relevant content
   
3. **Agent-Friendly JSON-LD++**
   - Every entity = structured data
   - FAQPage, HowTo, Recipe, Course schemas
   - AI agents understand everything

4. **Automatic Content Summarization**
   - AI generates TL;DR
   - Key takeaways extraction
   - Executive summaries for long posts

5. **Voice-Optimized Markup**
   - Structured for Alexa/Siri/Google
   - SpeakableSpecification schema
   - Read-aloud friendly formatting

### Tier 2: Static-First SEO

6. **Automatic Internal Linking**
   - AI suggests contextual links
   - Topic clustering
   - Pillar page detection
   
7. **Schema.org Maximizer**
   - Auto-detect content types
   - Generate all applicable schemas
   - Review snippets optimizer

8. **Table of Contents Generator**
   - Auto-TOC from headings
   - Anchor links (no JS)
   - Sticky TOC with CSS

9. **Reading Progress Indicator**
   - Pure CSS progress bar
   - Estimated reading time
   - No JavaScript required

10. **Breadcrumb Generator**
    - Auto-breadcrumbs from URL
    - Schema.org BreadcrumbList
    - Improves SEO + UX

### Tier 3: Content Quality Tools

11. **Content Freshness Auditor**
    - Detect outdated content
    - Suggest updates based on age
    - Auto-flag for review

12. **Broken Image Scanner**
    - Detect 404 images
    - Suggest replacements
    - R2 fallback integration

13. **Code Block Syntax Highlighter**
    - Server-side syntax highlighting
    - No Prism.js needed
    - Works without JS

14. **Accessibility Snapshot Tester**
    - Axe-core integration
    - Per-page a11y reports
    - Block builds on violations

15. **Performance Budget Reporter**
    - Track HTML/CSS/image sizes
    - Historical trends
    - Alert on budget violations

### Tier 4: Platform Integration

16. **Headless CMS Importer**
    - Import from Contentful, Sanity, Strapi
    - Keep as static files
    - Own your data

17. **Analytics Aggregator**
    - Combine Cloudflare, Plausible, Fathom
    - Unified dashboard
    - Export to CSV

18. **Email Newsletter Integration**
    - Export to Mailchimp, ConvertKit, Buttondown
    - RSS-to-email automation
    - Subscriber management

19. **Comment System (Static)**
    - GitHub Discussions integration
    - Webmentions support
    - No Disqus bloat

20. **Form Handler**
    - Netlify Forms, Formspree, Basin
    - Contact forms without backend
    - Platform-agnostic
