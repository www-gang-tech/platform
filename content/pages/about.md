# About Gang

GANG is an essentialist, AI-first static publishing platform built for teams who refuse to compromise on quality, accessibility, or performance.

## Our Philosophy

Every website should be:

1. **Fast** \- Sub\-2\.5s LCP\, zero client\-side JavaScript
2. **Accessible** \- WCAG 2\.2 AA minimum\, no exceptions
3. **Honest** \- Semantic HTML\, valid structured data
4. **Portable** \- Markdown \+ Git\, zero vendor lock\-in
